Creating This Website

As it has taken me some time to research how to build this website to my liking, I would like to share this knowledge with anyone who cares.

First of all, the technologies:

  • Emacs Org mode: the Org Publish package
  • Hosting using Github Pages

This is as basic as it can get. The idea is as follows:

  1. write an Org Publish configuration file
  2. write your webpages in Org files
  3. the html files are generated through the Org Publish package
  4. lastly, the generated html files are put in the git repository that represents your Github Pages location.

The greatest help of all came from these two articles:

  • Building a Emacs Org-Mode Blog
  • Publishing Org-mode files to HTML

I really liked these resources:

  • Practical Typography — Highly recommended
  • Colormind
  • Webdesign in 4 minutes
